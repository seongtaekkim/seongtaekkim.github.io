### 스케쥴 구성


공휴일
- ics 파일 안받고 수동 입력 하는 것으로.
- ics를 작성하지 말고 구글캘린더에 작성하면, 반영하는 것으로,
	- 전일 설정 시 24시간 반영 되는 지 테스트가 필요함.
